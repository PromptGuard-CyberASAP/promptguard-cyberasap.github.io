02Why now

AI is moving from answering questions to taking actions.

When someone pastes a client detail into a chatbot, uploads a file, or lets an agent call a tool, sensitive information can leave the building. Older data-loss tools were built for email and file shares, so they cannot see most of these new paths. Teams in law, finance and insurance feel it first.

Prompts
Text typed into an assistant
Names, matter numbers and figures pasted straight into a chat box.
Uploads
Files sent for analysis
Contracts and statements handed over for a summary or a redraft.
Tool calls
Agents reaching into systems
An agent that plans a task and pulls data from files, APIs and databases.
Agent to agent
One agent passing work to another
Information handed between collaborating agents, with no human in the loop.
